# 制造业分析模板

> 当 `industry-analysis` 将行业类型判断为"制造业"时加载此模板。
> 适用场景：有物理产品、制造工艺、上中下游供应链的行业。

## 产业链分析框架

```
上游（材料与元件）            中游（制造与集成）             下游（渠道与用户）
┌─────────────────┐       ┌──────────────────────┐       ┌────────────────────┐
│ 原材料           │       │ 核心工艺/制造         │       │ 终端市场/用户      │
│ 核心零部件       │ ───→  │ 组装/集成            │ ───→  │ 渠道/经销商        │
│ 设备/工具        │       │ QC/良率控制           │       │ 售后/服务          │
└─────────────────┘       └──────────────────────┘       └────────────────────┘
```

## 稀缺性控制点分析

**核心问题：利润流向哪个环节？**

| 环节 | 稀缺性（高/中/低） | 被谁控制 | 可替代性 |
|------|-------------------|---------|---------|
| 原材料/核心材料 | | | |
| 核心制造工艺 | | | |
| 关键零部件 | | | |
| 渠道/品牌 | | | |
| 技术/IP/专利 | | | |

**制造业的利润通常流向：**
- 供应受限的上游材料（如硫化锂）
- 高壁垒的核心工艺（如良率控制）
- 不可替代的品牌/渠道

## 超前指标方向

| 类别 | 具体指标示例 |
|------|-------------|
| **上游材料** | 关键材料价格、产能扩张计划 |
| **制造端** | 良率提升、产能利用率、设备出货量 |
| **产品端** | 成本下降曲线（如$/Wh）、性能指标提升（如MARD） |
| **政策端** | 补贴力度、关税变化、行业标准 |

## S曲线定位

```
性能/成本
    ↑
    │             现行技术（成熟期）← 指标：增速放缓、成本触底
    │    ────────
    │                   新技术（导入/成长期）← 指标：增速加速
    │                        ────
    │
    └──────────────────────────────────→ 时间
```

## 竞争分析

- **市场份额**：前3名占比？行业集中度（CR3/CR5）
- **主要玩家战略**：成本领先 vs 差异化 vs 聚焦
- **进入壁垒**：法规、资金、技术、渠道
